4. Spacing (Margin and Padding)
Margin

    m-{side}-{breakpoint}-{size}
        All sides: m-3
        Top: mt-3
        Right: mr-3
        Bottom: mb-3
        Left: ml-3
        Horizontal (left & right): mx-3
        Vertical (top & bottom): my-3
        Breakpoints: m-md-3, mt-lg-3, etc.

Padding

    p-{side}-{breakpoint}-{size}
        All sides: p-3
        Top: pt-3
        Right: pr-3
        Bottom: pb-3
        Left: pl-3
        Horizontal (left & right): px-3
        Vertical (top & bottom): py-3
        Breakpoints: p-md-3, pt-lg-3, etc.

Spacing Sizes

    0: m-0, p-0
    1: m-1, p-1
    2: m-2, p-2
    3: m-3, p-3
    4: m-4, p-4
    5: m-5, p-5
    Auto: m-auto

18. Responsive Design Utilities

Bootstrap 5 includes responsive utilities for controlling visibility and layout across different screen sizes:

    Responsive Breakpoints: sm, md, lg, xl, xxl
    Visibility Classes: d-none d-{breakpoint}-block
    Spacing Classes: m-{breakpoint}-{size}, p-{breakpoint}-{size}
