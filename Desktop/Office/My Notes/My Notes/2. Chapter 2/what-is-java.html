What Is Java:-

⇒ Defination :-

1. Programming Language :- Java is High Level Programming Language. It drives much of its syntax from C and C++, But it has fewer low-level facilities than C and C++.

2. Platform :- Java is a Platform because it has its own run time environment that is JVM(Java Virtual Machine). JVM provides a platform which accepts the byte code and executes on the machine.
