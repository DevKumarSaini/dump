Literals In Java:-

⇒ Definition :- A Literal is a sequence of characters (digits, letters, and other characters) that represent constant values to be stored in variables or simply say, the value assigned to a variable is known as literal.

For example: 
                    
                        
 int a=10;    // here "10" is literal
                        
 char c='a';    // here "a" is literal
                        
 String name="deepak";    // here "deepak" is literal
