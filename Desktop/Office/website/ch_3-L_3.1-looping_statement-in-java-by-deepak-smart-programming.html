Looping statements / Iterations / Control flow

Looping Statements

Looping statements, also known as loops, are control structures used in programming to execute a block of code repeatedly based on certain conditions. They allow you to perform repetitive tasks without writing the same code multiple times. There are several types of looping statements, each suited for different scenarios

Types of looping statements
  1. for loop
  2. while loop
  3. do while loop
  4. for each loop
Java For loop
  • Definition

    A for loop is a fundamental control structure in programming that allows you to execute a block of code repeatedly with a controlled number of iterations. It is commonly used when you know in advance how many times you want to execute a block of code. Let’s break down the concept in detail.

  • Detailed Working
    1. Syntax 
      for (any java valid statement; condition; increment/decrement) {
 for loop body
}
    2. Component
      • Initialization An expression executed once before the loop starts. Typically used to initialize a loop counter.
      • Condition An expression evaluated before each iteration. The loop continues as long as this condition evaluates to true.
      • Update An expression executed after each iteration. Typically used to update the loop counter.
    3. flow
      • Execute the initialization expression.
      • Evaluate the condition expression.
      • If the condition is true, execute the code block inside the for loop.
      • Execute the update expression.
      • Repeat steps 2-4 until the condition evaluates to false.
    4. Example of Different condition
      • for (int i = 0; i < 5; i++) // Iterates from i = 0 to i = 4
      • for (int j = 10; j > 0; j--) // Iterates from j = 10 to j = 1
      • for (int k = 0; k <= 100; k += 10) // Iterates from k = 0 to k = 100, in steps of 10
  • E-R Diagram
    An ER diagram is not applicable to the for loop since it pertains to control flow rather than entity relationships.
  • Program
    public class ForLoopExample {
    public static void main(String[] args) {
        // Using a for loop to print numbers from 1 to 5
        for (int i = 1; i <= 5; i++) {
            System.out.println("Number: " + i);
        }
        
        System.out.println("Loop finished.");
    }
}
    Output
    Number: 1
Number: 2
Number: 3
Number: 4
Number: 5
Loop finished.
  • Real-life Example
    • Initialization Start with a counter, e.g., i = 1.
    • Condition Continue as long as the counter is less than or equal to 5 (i <= 5).
    • Update Increment the counter by 1 after each iteration (i++).
    • Action Print the current value of the counter.
  • Use case
    • Iterating over collections Such as arrays, lists, or ranges.
    • Repeating a task a certain number of times For instance, printing numbers, calculating factorials, or performing a task a fixed number of times.
    • Indexing Useful for accessing elements in arrays or lists.

Java while loop
  • Definition

    The while loop in Java is a control flow statement that repeatedly executes a block of code as long as a specified condition evaluates to true. It is used for scenarios where the number of iterations is not known beforehand and depends on a condition that is evaluated before each iteration.

  • Detailed Working
    1. Syntax 
      while (condition) {
// code to be executed as long as the condition is true
}
    2. Component
      • Condition An expression evaluated before each iteration. The loop continues executing as long as this condition evaluates to true.
      • Code Block The block of code that executes on each iteration as long as the condition is true.
    3. Flow
      • Evaluate the condition expression.
      • If the condition is true, execute the code block inside the while loop.
      • After executing the code block, re-evaluate the condition.
      • Repeat steps 2-3 until the condition evaluates to false.
    4. Example of Different Condition
      • while (x < 10) // Continues as long as x is less than 10
      • while (isRunning) // Continues as long as isRunning is true
      • while (temperature != 20) // Continues as long as temperature is not equal to 20
  • E-R Diagram

    An ER diagram is not applicable to the while loop since it pertains to control flow rather than entity relationships.

  • Program
    public class WhileLoopExample {
    public static void main(String[] args) {
        int count = 1;

        // Using a while loop to print numbers from 1 to 5
        while (count <= 5) {
            System.out.println("Count: " + count);
            count++; // Update the count
        }
        
        System.out.println("Loop finished.");
    }
}
  • Real-life Example
    • Condition As long as you have more tasks to complete (tasksRemaining > 0).
    • Action If the condition is true, perform the task and update the count of remaining tasks. If the condition is false, stop performing tasks.
  • Use case
    • Prompt Until Valid Input You can use a while loop to repeatedly prompt a user for input until they provide valid data. For example, asking for a number within a specific range until they provide one.
    • File Processing When reading from a file, you can use a while loop to process each line or chunk of data until the end of the file is reached.
    • Retry Mechanism You can use a while loop to implement retry mechanisms, such as attempting to perform an operation and retrying until it succeeds or a timeout occurs.
    • Games or Real-Time Applications In games or real-time systems, a while loop can be used to run the main game loop, updating the game state and rendering frames continuously.
    • Processing Elements Until Condition When working with collections or data streams, a while loop can process elements until a certain condition is met.
    • Generating Series You can use a while loop to generate a series of values or accumulate results based on certain conditions.

Do while loop
  • Definition

    The do-while loop in Java is a control flow statement that executes a block of code at least once and then repeatedly executes it as long as a specified condition evaluates to true. This loop guarantees that the code block will be executed at least once before the condition is tested.

  • Detailed Working
    1. Syntax 
      do {
    // code to be executed
    } while (condition);
    2. Component
      • Code Block The block of code that executes once before the condition is evaluated and repeatedly as long as the condition remains true.
      • Condition An expression evaluated after each execution of the code block. The loop continues executing as long as this condition evaluates to true.
    3. Flow
      • Execute the code block inside the do part.
      • Evaluate the condition expression.
      • If the condition is true, repeat the execution of the code block.
      • If the condition is false, exit the loop and continue with the rest of the program.
    4. Example of Different Condition 
      • do {
    // code to be executed
} while (x < 10); // Continues as long as x is less than 10 
      • do {
    // code to be executed
} while (userInput.equals("yes")); // Continues as long as userInput is "yes"
      • do {
    // code to be executed
} while (temperature != 20); // Continues as long as temperature is not equal to 20
  • E-R Diagram

    An ER diagram is not applicable to the do-while loop since it pertains to control flow rather than entity relationships.

  • Program
    public class DoWhileLoopExample {
    public static void main(String[] args) {
        int count = 1;

        // Using a do-while loop to print numbers from 1 to 5
        do {
            System.out.println("Count: " + count);
            count++; // Update the count
        } while (count <= 5);
        
        System.out.println("Loop finished.");
    }
}
  • Real-life Example
    • Condition Keep asking for user input until the user provides a valid response (userInput.equals("yes")).
    • Action Prompt the user and process their response. The loop will run at least once regardless of the initial condition.
  • Use case
    • Menu-Driven Programs A common use case for a do-while loop is in menu-driven applications where you want to show a menu to the user and perform an action based on user input. You might want to keep showing the menu and executing user commands until the user chooses to exit.
    • User Input Validation When you need to ensure that user input meets certain criteria, a do-while loop can be used to repeatedly prompt the user until valid input is received.
    • Retry Logic If a task needs to be retried upon failure, such as retrying a connection to a server or reattempting a file operation, a do-while loop ensures that at least one attempt is made.

For each loop
  • Definition

    The for-each loop in Java is a control flow statement that iterates over elements in arrays or collections (such as ArrayList, HashSet, etc.). It provides a simplified way to loop through each element without the need for an explicit counter or index.

  • Detailed Working
    1. Syntax 
      for (type item : collection) {
    // code to be executed for each item
}
    2. Component
      • type The data type of the elements in the collection or array.
      • item A variable that represents the current element in each iteration of the loop.
      • collection The array or collection that you are iterating over.
    3. Flow
      • Retrieve the first element from the collection or array.
      • Execute the code block using this element.
      • Move to the next element and repeat step 2.
      • Continue until all elements have been processed.
    4. Example of Different Condition 
      // Example with an array
int[] numbers = {1, 2, 3, 4, 5};
for (int number : numbers) {
    System.out.println("Number: " + number);
}

// Example with a list
List names = Arrays.asList("Alice", "Bob", "Charlie");
for (String name : names) {
    System.out.println("Name: " + name);
}
  • E-R Diagram
    An ER diagram is not applicable to the for-each loop since it pertains to control flow rather than entity relationships.
  • Program
    import java.util.Arrays;
import java.util.List;

public class ForEachLoopExample {
    public static void main(String[] args) {
        // Using a for-each loop to iterate over an array
        String[] fruits = {"Apple", "Banana", "Cherry"};
        
        for (String fruit : fruits) {
            System.out.println("Fruit: " + fruit);
        }
        
        // Using a for-each loop to iterate over a list
        List numbers = Arrays.asList(10, 20, 30, 40, 50);
        
        for (int number : numbers) {
            System.out.println("Number: " + number);
        }
    }
}
  • Real-life Example
    • Collection A list of books on a shelf.
    • Action Display each book title. You don’t need to worry about the position of each book; you just iterate through the list and handle each book.
  • Use case
    • Iterating Over Arrays or Lists The for-each loop is ideal for iterating over arrays or lists where you want to perform an operation on each element without needing to use an index.
    • Processing Elements of a Collection When working with collections like lists, sets, or dictionaries (in some languages), the for-each loop allows you to process each element in a clean and readable manner.
    • Modifying Elements in a Collection In some cases, you might need to modify elements of a collection. Note that while for-each loops make it easy to access elements, modifying elements depends on whether the loop provides access to references or copies.
    • Iterating Over Map/Dictionary Entries In some languages, for-each loops can also be used to iterate over key-value pairs in a map or dictionary.
