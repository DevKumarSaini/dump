Ternary

Ternary Operator

The ternary operator in Java is a concise way to perform conditional evaluations. It allows you to return one of two values based on a condition. The operator is often used as a shorthand for if-else statements. The ternary operator is represented by the ? and : symbols

Detailed Working
Syntax
result = (condition) ? valueIfTrue : valueIfFalse; // 'result', 'condition', 'valueIfTrue', 'valueIfFalse' are identifiers
  • Condition: The expression that is evaluated. If it evaluates to true, the valueIfTrue is assigned to result; otherwise, valueIfFalse is assigned. (condition, valueIfTrue, valueIfFalse are identifiers)
  • Value If True: The value that is assigned to result if the condition evaluates to true. (valueIfTrue is an identifier)
  • Value If False: The value that is assigned to result if the condition evaluates to false. (valueIfFalse is an identifier)
  • Result: The variable where the result of the ternary operation is stored. (result is an identifier)
Flow
  1. Evaluate the condition. (condition is an identifier)
  2. If the condition is true, assign the valueIfTrue to the result. (valueIfTrue, result are identifiers)
  3. If the condition is false, assign the valueIfFalse to the result. (valueIfFalse, result are identifiers)
Example
int a = 10; // 'a' is a variable identifier
String result = (a > 5) ? "Greater than 5" : "less than 5"; 
System.out.println(result); // 'System', 'out', 'println', 'result' are identifiers
Explanation of Identifiers
  • Variable Identifier: a, result
  • Condition Identifier: a > 5
  • Values: "Greater than 5", "less than 5"
  • Output Identifiers: System, out, println
Real-life Example

Condition: Based on whether a number is greater than 5 (e.g., a > 5). (a, 5 are identifiers)

Action: If a is greater than 5, the result is "Greater than 5". If not, the result is "less than 5". (result, "Greater than 5", "less than 5" are identifiers)

Some more example of ( Ternary ) statements
  • Basic Comparison 
    public class Main {
    public static void main(String[] args) {
        int number = -10;
        String result = (number >= 0) ? "Positive" : "Negative";
        System.out.println(result); // Outputs: Negative
    }
}
    Output
    Negative
  • Check for Voting Eligibility 
    public class Main {
    public static void main(String[] args) {
        int age = 22;
        String canVote = (age >= 18) ? "Eligible to vote" : "Not eligible to vote";
        System.out.println(canVote); // Outputs: Eligible to vote
    }
}
    Output
    Eligible to vote
  • Find the Maximum of two number 
    public class Main {
    public static void main(String[] args) {
        int a = 15, b = 20;
        int max = (a > b) ? a : b;
        System.out.println("Maximum: " + max); // Outputs: Maximum: 20
    }
}
    Output
    Maximum: 20
  • String Length Check 
    public class Main {
    public static void main(String[] args) {
        String str = "Java";
        String lengthCheck = (str.length() > 5) ? "Long String" : "Short String";
        System.out.println(lengthCheck); // Outputs: Short String
    }
}
    Output
    Short String
  • Discount Calculation 
    public class Main {
    public static void main(String[] args) {
        double price = 150.0;
        double discount = (price > 100) ? price * 0.1 : 0;
        System.out.println("Discount: " + discount); // Outputs: Discount: 15.0
    }
}
    Output
    Discount: 15.0
  • Ternary Statement inside if-else 
    public class TernaryUsingIf {
    public static void main(String[] args) {
        String email = "checkprogramming@gmail.com";

        // Using an if statement to decide whether to print the message
        if (email.contains("@gmail.com") ? true : false) {
        System.out.println("This is valid email."); // Output valid email
        } else {
        System.out.println("This is not valid email.");
        }
    }
}
    Output
    This is valid email.
