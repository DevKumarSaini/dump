WAP to Check for Armstrong Number in Java 

import java.util.Scanner;
class class ArmstrongNumber
{
    public static void main(String args[])
    {
        Scanner s=new Scanner(System.in);
        System.out.println("Enter The Number : ");
        int no=s.nextInt();
        int t1=no;
        int leng=0;
        while(t1 != 0)
        {
            leng=leng+1;
            t1=t1/10;
        }
        	
        int t2=no;
        int arm=0;
        int rem;
        while(t2 != 0)
        {
            int mul=1;
            rem=t2%10;
            for(int i=1;i<=leng;i++)
            {
                mul=mul*rem;
            }
            arm=arm+mul;
            t2=t2/10;
        }
        if(arm==no)
        {
            System.out.println(no+" is armstrong number");
        }
        else
        {
            System.out.println(no+" is not armstrong number");
        }
    }
}
Output
Enter The Number :
153
153 is armstrong number

WAP To Create Calculator Using switch case In Java.

import java.util.Scanner;
class Test
{
    public static void main(String[] args)
    {
        String yn;
        do
        {
            Scanner s=new Scanner(System.in);
            System.out.println("Enter First No. ");
            int no1=s.nextInt();
            System.out.println("Enter Second No. ");
            int no2=s.nextInt();
            System.out.println("Select Symbol (+,-,*,/)");
            String sym=s.next();
            int res;
            switch(sym)
            {
                case "+": res=no1+no2;
                    System.out.println("Addition is : "+res);
                    break;
                case "-": res=no1-no2;
                    System.out.println("Subtraction is : "+res);
                    break;
                case "*": res=no1*no2;
                    System.out.println("Multiplication is : "+res);
                    break;
                case "/": res=no1/no2;
                    System.out.println("Division is : "+res);
                    break;
                default : System.out.println("Invalid Symbol");
                    break;
            }
            System.out.println("Do u want to continue (Press y for Yes and n for No)");
            yn=s.next();
        }while(yn.equals("y") || yn.equals("Y"));
    }
}
Output
Enter First No.
5
Enter Second No.
5
Select Symbol (+,-,*,/)
+
Addition is : 10
Do u want to continue (Press y for Yes and n for No)
n

WAP to find factorial using Recursive approach.

import java.util.Scanner;

class FactorialByRecursion
{
    static int fact=1;
    public static void main(String args[])
    {
        Scanner s=new Scanner(System.in);
        System.out.println("Enter the no : ");
        int no=s.nextInt();
        FactorialByRecursion ob=new FactorialByRecursion();
        ob.calcFact(no);
        System.out.println("factorial of "+no+" is "+fact);
    }
    void calcFact(int no)
    {
        if(no>=1)
        {
            fact=fact*no;
            calcFact(no-1);
        }
    }
}
Output
Enter the no
6
factorial of 6 is 720

WAP To Reverse A String in Java

class ReverseAString
{
    public static void main(String args[])
    {
        String name="vinay";
        int leng=name.length();
        String rev="";
        for(int i=leng-1; i>=0; i--)

        System.out.println("Reverse of  "+name+" is "+rev);
    }
