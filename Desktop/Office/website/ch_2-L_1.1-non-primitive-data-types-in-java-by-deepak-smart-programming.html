Non - Primitive Data Types in Java

Datatype in Java

In Java, data types define the kind of data that can be stored and manipulated within a program. They specify the size and type of values that can be stored in variables.

Points to remember
  1. Every Variable Has a Defined Type: In Java, every variable, expression, and operation has a specific, predefined type (like int, float, String, etc.)
  2. Strict Type Checking: Java ensures that the types you use are consistent and compatible with each other. For example, if a variable is declared as an int, you can't just assign a String to it.
  3. No Automatic Type Conversions: Unlike some other programming languages, Java doesn’t automatically convert between incompatible types. For instance, you can't directly assign a float to an int without explicitly converting it.
  4. Compiler Enforces Type Safety: During compilation, the Java compiler checks all code to make sure the types match. If there's a mismatch, the compiler will throw an error, and the code won't compile until the issue is fixed.


Java is a statically-typed programming language. It means, all variables must be declared before its use. That is why we need to declare variable's type and name.
