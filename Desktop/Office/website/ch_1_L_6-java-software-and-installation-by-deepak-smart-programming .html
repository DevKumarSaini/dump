Java Software installation

Steps to Install Java software

  1. Understanding the Java Development Kit (JDK)
    • Java Runtime Environment (JRE): Provides libraries, Java Virtual Machine (JVM), and other components to run Java applications.
    • Development Tools: Includes tools such as the Java compiler (javac), the Java debugger (jdb), and other tools needed for Java development.
  2. Downloading the JDK
    1. Visit the Oracle JDK download page: Go to Oracle’s JDK download page.
    2. Choose the JDK version: Select the version you need (e.g., JDK 21). Make sure it matches your operating system.
    3. Download the installer: Click the appropriate download link for your operating system (Windows, macOS, or Linux).
  3. Installing the JDK
    • For Windows:
      1. Run the installer: Double-click the downloaded .exe file.
      2. Follow the installation wizard:
        • Click “Next” on the setup wizard.
        • Choose the installation directory. The default is usually fine, but you can change it if needed.
        • Click “Install” and wait for the installation to complete.
        • Click “Finish” when done.
      3. Set up environment variables:
        • Open the RUN dialog box by pressing win key + R, type sysdm.cpl then hit enter.
        • In the System Properties window, click “Advance Tab.”
        • In the System Properties window, click “Environment Variables” under "Advance" tab.
        • Then click on new under System-variables .
        • In new system variable block type JAVA_HOME in variable name.
        • Then copy your directory location address using below following given steps.
        • Then past in Value variable and press Ok
        • After all the steps done please make sure all the entry of JDK is done in your system and displayed here.
      4. Verify installation: Open Command Prompt and run java -version and javac -version to ensure both Java and the Java compiler are installed correctly.
        • Step 1: press win key + R to open run dialog box and then type cmd.
        • Step 2: Once after opening command prompt simply type java (if java information is displayed it means java is successfully installed and configured) else (if java is not configured properly it would displayed 'java' is not recognized as an internal or external command, operable program or batch file. )
        • Step 3: Or you can also verify java configuration by checking java version. To check java version just simply type java -version in cmd if (it shows the version of java you configured in system means java configuration is successful else not.)
    • For macOS:
      1. Run the installer: Double-click the downloaded .dmg file and run the .pkg installer.
      2. Follow the installation prompts: Follow the on-screen instructions to complete the installation.
      3. Set up environment variables:
        • Open Terminal.
        • Edit your profile file (e.g., .zshrc, .bash_profile, or .bashrc depending on your shell) to include export JAVA_HOME=$(/usr/libexec/java_home).
        • Run source ~/.zshrc or source ~/.bash_profile to apply the changes.
      4. Verify installation: Open Terminal and run java -version and javac -version.
    • For Linux:
      1. Use the package manager:
        • For Debian-based distributions (e.g., Ubuntu), you can use: 
          sudo apt update
                        sudo apt install openjdk-21-jdk
        • For Red Hat-based distributions (e.g., Fedora), use: 
          sudo dnf install java-21-openjdk-devel
      2. Set up environment variables:
        • Open Terminal.
        • Edit your profile file (e.g., .bashrc or .profile) to include: 
          export JAVA_HOME=$(dirname $(dirname $(readlink -f $(which javac))))
                        export PATH=$JAVA_HOME/bin:$PATH
        • Run source ~/.bashrc or source ~/.profile to apply the changes.
      3. Verify installation: Open Terminal and run java -version and javac -version.
  4. Additional Tips
    • Integrated Development Environment (IDE): Consider installing an IDE like IntelliJ IDEA, Eclipse, or NetBeans to simplify Java development.
    • Updating JDK: Periodically check for updates to ensure you have the latest features and security patches.
