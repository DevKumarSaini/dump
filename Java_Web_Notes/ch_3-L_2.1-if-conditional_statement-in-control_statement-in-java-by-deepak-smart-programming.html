Conditional Statements

Conditional Statements / Decision Making Statements / Selection Statements

Conditional statements are fundamental constructs in programming that allow a program to make decisions and execute different code blocks based on certain conditions. In Java, conditional statements control the flow of execution depending on whether a condition evaluates to true or false. This enables the program to perform different actions based on varying inputs or states.

  1. Types of Conditional statements
    • if
    • if-else
    • else-if
    • else-if-ladder

  1. Java if statement

    1. Definition

      The if statement in Java is a control flow statement that allows the execution of a block of code if a specified condition evaluates to true. It is used to perform conditional operations and make decisions in the code.

    2. Detailed Working
      1. Syntax 
        if (condition) {
// code to be executed if condition is true
}
      2. Component
        • Condition An expression that evaluates to a boolean value (true or false).
        • Code Block The block of code that executes if the condition is true.
      3. flow
        • Evaluate the condition.
        • If condition is true, execute the code block inside the if statement.
        • If condition is false, skip the code block and continue with the rest of the program.
      4. Example of Different Condition
        • if (x > 10)
        • if (isLoggedIn == true)
        • if (temperature < 0)
    3. E-R Diagram

      An ER diagram is not applicable to the if statement since it pertains to control flow rather than entity relationships.

    4. Program
      public IfStatementExample {
    public static void main(String[] args) {
        int age = 18;

        // Using an if statement to check if the person eligible to vote
        if (age >= 18) {
            System.out.println("You are eligible to vote.");
        }

        System.out.println("This line always executes.");
    }
}
      Output
      You are eligible to vote.
This line always executes.
    5. Real-life Example
      • Condition If it is sunny (e.g., weather == "sunny").
      • Action If the condition is true, you go for a walk. If the condition is false, you stay indoors.

  2. Java if-else Statement

    1. Definition

      The if-else statement in Java is a control flow statement that allows the execution of one block of code if a specified condition evaluates to true, and another block if the condition evaluates to false. It is used to perform conditional operations and make decisions in the code.

    2. Detailed Working
      1. Syntax 
        if (condition) {
// code to be executed if condition is true
} else {
// code to be executed if condition is false
}
      2. Component
        • Condition An expression that evaluates to a boolean value (true or false).
        • True Block The block of code that executes if the condition is true.
        • False Block The block of code that executes if the condition is false.
      3. Flow
        • Evaluate the condition.
        • If the condition is true, execute the code block inside the `if` statement.
        • If the condition is false, execute the code block inside the `else` statement.
      4. Example of Different Conditions
        • if (temperature > 30)
        • if (isWeekend == true)
        • if (score >= 50)
    3. E-R Diagram

      An ER diagram is not applicable to the if-else statement since it pertains to control flow rather than entity relationships.

    4. Program
      public class IfElseExample {
    public static void main(String[] args) {
        int score = 75;
        // Using an if-else statement to check if the person passed or failed
        if (score >= 60) {
            System.out.println("You passed.");
        } else {
            System.out.println("You failed.");
        }
        System.out.println("This line always executes.");
    }
}
      Output
      You passed. This line always executes.
    5. Real-life Example
      • Condition If the temperature is above 30°C (e.g., temperature > 30).
      • Action If the condition is true, you go to the beach. If the condition is false, you stay indoors.

  3. Java else-if Statement

    1. Definition

      The else-if statement in Java allows you to test multiple conditions sequentially. It is an extension of the if-else statement, providing a way to specify additional conditions and corresponding actions. This structure is useful when you have more than two conditions to evaluate.

    2. Detailed Working
      1. Syntax 
        if (condition1) {
    // code to be executed if condition1 is true
    } else if (condition2) {
    // code to be executed if condition2 is true
    } else if (condition3) {
    // code to be executed if condition3 is true
    } else {
    // code to be executed if none of the above conditions are true
    }
      2. Component
        • Condition An expression that evaluates to a boolean value (true or false). Multiple conditions can be evaluated in sequence.
        • True Block The block of code that executes if a specific condition is true.
        • Else Block The block of code that executes if none of the specified conditions are true.
      3. Flow
        • Evaluate the first condition.
        • If the first condition is true, execute the corresponding code block and skip the remaining conditions.
        • If the first condition is false, evaluate the next condition in the `else-if` statement.
        • Continue evaluating until a true condition is found or all conditions are checked.
        • If none of the conditions are true, execute the code block inside the `else` statement.
      4. Example of Different Conditions
        • if (temperature > 30)
        • else if (temperature > 20)
        • else if (temperature > 10)
        • else
    3. E-R Diagram

      An ER diagram is not applicable to the else-if statement since it pertains to control flow rather than entity relationships.

    4. Program
      public class ElseIfExample { public static void main(String[] args) { int score = 85;

    // Using an else-if statement to determine the grade
    if (score >= 90) {
    System.out.println("Grade: A");
    } else if (score >= 80) {
    System.out.println("Grade: B");
    } else if (score >= 70) {
    System.out.println("Grade: C");
    } else {
    System.out.println("Grade: D");
    }
    System.out.println("This line always executes.");
    }
}
      Output
      Grade: B
This line always executes.
    5. Real-life Example
      • Condition Based on the time of day (e.g., if time < 12).
      • Action If the condition is true, you say "Good Morning". If false and the time is < 18, you say "Good Afternoon" . If false and the time is < 24, you say "Good Evening" . Otherwise, you say "Good Night" .

  4. Java else-if Ladder

    1. Definition

      The else-if ladder is an extension of the basic `if-else` statement in Java that allows you to test multiple conditions in a sequence. It provides a way to handle several conditions by evaluating each one in order until one of them evaluates to true. If none of the conditions are true, the `else` block is executed, if present.

    2. Detailed Working
      1. Syntax 
        if (condition1) {
// code to be executed if condition1 is true
} else if (condition2) {
// code to be executed if condition2 is true
} else if (condition3) {
// code to be executed if condition3 is true
} else {
// code to be executed if none of the above conditions are true
}
      2. Component
        • Condition An expression that evaluates to a boolean value (true or false). Multiple conditions are evaluated in sequence.
        • True Block The block of code that executes if a specific condition is true.
        • Else Block The block of code that executes if none of the specified conditions are true.
      3. Flow
        • Evaluate the first condition.
        • If the first condition is true, execute the corresponding code block and skip the remaining conditions.
        • If the first condition is false, evaluate the next condition in the `else-if` ladder.
        • Continue evaluating each `else-if` condition in sequence.
        • If none of the conditions are true, execute the code block inside the `else` statement, if present.
      4. Example of Different Conditions
        • if (temperature > 35)
        • else if (temperature > 25)
        • else if (temperature > 15)
        • else if (temperature > 5)
        • else
    3. E-R Diagram

      An ER diagram is not applicable to the else-if ladder since it pertains to control flow rather than entity relationships.

    4. Program
      public class ElseIfLadderExample {
        public static void main(String[] args) {
            int temperature = 20;

            // Using an else-if ladder to determine the weather condition
            if (temperature > 35) {
                System.out.println("It's very hot outside.");
            } else if (temperature > 25) {
                System.out.println("It's warm outside.");
            } else if (temperature > 15) {
                System.out.println("It's cool outside.");
            } else if (temperature > 5) {
                System.out.println("It's cold outside.");
            } else {
                System.out.println("It's freezing outside.");
            }

            System.out.println("This line always executes.");
    }
}
      Output
      It's cool outside.
This line always executes.
    5. Real-life Example
      • Condition Based on the score achieved in an exam (e.g., if score >= 90).
      • Action If the score is greater than or equal to 90, you get an "A". If the score is between 80 and 89, you get a "B". If the score is between 70 and 79, you get a "C". If the score is between 60 and 69, you get a "D". Otherwise, you get an "F".
